# Evasão Escolar 

## Resumo 
A evasão escolar é um desafio crítico que afeta não apenas o desempenho educacional dos estudantes, mas também tem implicações profundas na sociedade e na economia. Este projeto utiliza técnicas de ciência de dados, análise estatística e visualização de dados para investigar o problema da evasão escolar, e como podemos predizer, através de outras variáveis relacionáveis, o que leva os alunos a uma maior evasão ou não. Utilizaremos o conjunto de dados [Students' Academic Performance Dataset](https://www.kaggle.com/datasets/aljarah/xAPI-Edu-Data) para elaboração do projeto.
